Project Title

자바의 특징으로는 c++에 비해 단순하며 분산 환경 및 보안성을 지원하고 컴파일을 거쳐야 한다. 또 컴파일을 통해 생성된 Class 파일을 가상 머신을 통해 실행해야된다.

객체지향 프로그래밍의 구성요소로는 객체와 클래스, 메시지가 있다. 객체는 세상에 존재하는 모든것들을 말하며 예로는 사람, 휴대폰, 사물함, 인터넷 등으로 들 수있다. 클래스는 객체의 틀이다. 메시지는 객체 간의 통신을 말한다.

메소드 오버로딩

오버로딩은 한 클래스 내에서의 메소드를 중복해서 작성하는 것이다. 서로 매개 변수의 타입이나 개수가 다른 여러 개의 메소드가 같은 이름으로 작성되는 것을 말한다. 메소드명이 같은 여러 대의 메소드를 중복 선언하여 사용의 편리성을 향상 시키는것이 오버로딩의 목적이다.

메소드 오버라이딩

메소드 오버라이딩은 클래스 상속 상황에서 부모 class의 멤버를 자식 class에서 상속받았지 만 자식 class에서 해당 멤버의 내용을 수정하여 자식 class 객체에서 적용한다. 메소드 오버라이딩은 부모 class의 정의에는 영향을 주지 않는다.
